# finalYearProject
Completed a Final Year Project as part of my undergraduate degree in Business Information Systems at University College Cork that ran for the full academic year of my final year of study.
Developed a full stack Android mobile application written in Java, PHP and SQL, using XAMPP and phpMyAdmin to integrate a MySQL database, completed using agile development methodologies. Achieved a 1H1.
Android application was named 'ReRecruit' and provided 'returners' (those who have taken extended career breaks' a tool to help them gain the confidence to return to the workforce and identify available opportunities at organisations actively hiring their demographic.
